# Comix Helper

**Comix Helper** is a userscript designed to enhance the reading experience on [Comix.to](https://comix.to). It enables WASD and Arrow Key navigation, fixes broken keyboard scrolling, and provides a customizable settings interface for scroll behavior.

## üöÄ Features

* **Fixes Keyboard Scrolling:** Use `W`/ `S` or `Arrow Keys` to scroll.

* **Chapter Switching:** Quickly jump to the next or previous chapter using `A` / `D` or Left/Right arrows.

* **Customizable Settings:** Adjust scroll speed and animation behavior (Instant vs. Smooth).

* **Toggleable:** Quickly enable or disable the script using a hotkey.

## üì• Installation

1. **Install a Userscript Manager:**

   * [Tampermonkey](https://www.tampermonkey.net/) (**Required** - This script has only been tested with Tampermonkey).

   * *Note: Other managers like Violentmonkey or Greasemonkey might work, but they have not been tested.*

2. **Install the Script:**

   * [**Click Here to Install**](https://raw.githubusercontent.com/Fear3d/Comix-Helper/main/Comix-Helper.user.js)

3. **Visit Comix.to:** Open any chapter and the script will automatically activate.

## üéÆ Controls

| **Key** | **Action** | 
| **W** / **Up Arrow** | Scroll Up | 
| **S** / **Down Arrow** | Scroll Down | 
| **A** / **Left Arrow** | Previous Chapter | 
| **D** / **Right Arrow** | Next Chapter | 
| **Page Up** | Scroll Up (Large Step) | 
| **Page Down** | Scroll Down (Large Step) | 
| **?** (Shift + /) | Show Help Menu | 
| **Ctrl + Shift + F** | Toggle Hotkeys On/Off | 

## ‚öôÔ∏è Configuration

You can customize the scrolling speed and animation behavior.

1. Click on the Tampermonkey icon on your browser toolbar.

2. Select **"Configure Settings"**.

3. A modal will appear allowing you to adjust:

   * **Step Scroll Speed:** How far the screen moves with W/S and arrow keys.

   * **Page Scroll Amount:** How far the screen moves with PgUp/PgDn.

   * **Behaviors:** Choose between `Auto`, `Instant`, or `Smooth` scrolling.
